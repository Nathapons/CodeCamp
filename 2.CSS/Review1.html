CSS syntax ประกอบด้วยอะไรบ้าง
  1. Selector คือ องค์ประกอบ HTML ที่คุณต้องการจัดรูปแบบ
  2. Declation คือ บล็อกการประกาศประกอบด้วยการประกาศอย่างน้อยหนึ่งรายการโดยคั่นด้วยอัฒภาค
    1. Property คือ ชื่อคุณสมบัติ CSS และค่าคั่นด้วยเครื่องหมายทวิภาค
    2. Value คือ ค่าที่เป็นตัวกำหนดปริมาณของ Property
 h1 {color: #ddd; font-size:20px}
  h1 = Selector
  {color: #ddd; font-size:20px} = Declation
  color, font-size = Properties
  #ddd, 20px = value
การเขียน CSS มีกี่แบบอะไรบ้าง 1. Inline: ภายใน HTML tag
2. External: นำเข้าไฟล์ CSS จากภายนอก
3. Internal: เขียนภายใน Element HTML
 1. <h1 style="color:blue;text-align:center;">This is a heading</h1>
2. <link rel="stylesheet" href="mystyle.css">
3. body {background-color: lightblue;}
selector มีกี่แบบอะไรบ้าง 1. ID การกำหนดชื่อของ Elemet ได้แค่ 1 element เท่านั้น
2. Class การกำหนดชื่อของ Elemet ได้หลายelement เท่านั้น
3. Pseudo element การกำหนดเฉพาะบางส่วนของ element
4. Pseudo class การกำหนดเฉพาะบางสถานะของ class
5.Combinators การกำหนดเงื่อนระหว่าง 2 selector
6. attribute selector การกำหนดการเลือกชื่อ attribute ใน element
combinator มีกี่แบบอะไรบ้าง 1. Descendant Selector: เลือกทั้งหมดขององค์ประกอบ tag
2. Child Selector: ตัวเลือกลูกจะเลือกองค์ประกอบทั้งหมดที่เป็นลูกขององค์ประกอบที่ระบุ
3. Adjacent Sibling: องค์ประกอบพี่น้องต้องมีองค์ประกอบหลักเดียวกันและติดกัน
4. General Sibling Selector: ตัวเลือกพี่น้องทั่วไปจะเลือกองค์ประกอบทั้งหมดที่เป็นพี่น้องขององค์ประกอบที่ระบุ
 1. div p{} tag p ทั้งหมดที่อยู่ใน div
2. div>p{} = tag div ที่เป็นลูกของ p
3. div + p{} ที่อยู่ล่าง div และติดกับ div เท่านั้น
4. p ~ ul{} = tag p ที่อยู่ล่าง div ทุก tag
property และ value คืออะไรแตกต่างกันอย่างไร Property จะเป็นตัวกำหนดคุณสมบัติของ tag แต่ value จะเป็นตัวกำหนดปริมาณของ property
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง 1. Property for content: color , size , font , text, list-style-type, Height, Width, background
2. Property for border : border, table
3. Property for padding : padding (top right buttom left)
4. Property for margin : margin (top right buttom left)
5. Property for element : display, Height, Width, Position, Float, Align, Flex, Grid
value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร 1. Absolute lenghts จะบอกในรูปแบบของหน่วย เช่น px cm
2. relative lenghts จะบอกในรูปแบบเป็นเท่าหรือเปอร์เซนต์
