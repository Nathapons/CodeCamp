Question Answer
value ใน display property มีอะไรบ้าง 1. Inline อยู่บรรททัดเดียวกัน ไม่สามารถปรับแต่ง ความกว้าง ความสูงได้
2. Block คล้าย tag <p> อยู่คนละบรรทัด
3. Content ทำให้ element นั้นกลายเป็น content ไม่ได้
4. Flex จัดรูปแบบ flex
5. Grid การจัดรูปแบบ grid
6. Run-in : block or inline ขี้นอยู่กับ เนื้อหา
7. Inline-(block, flex, grid, table)
2.grid และ flex แตกต่างกันอย่างไร grid คือ คุณสมบัติในรูปแบบแถวและคอลัมน์
flex คือ การกำหนดคุณสมบัต์ให้มีความยืดหยุ่น
3.justify and align แตกต่างกันอย่างไร justify จัดรูปข้อความแบบแนวตั้ง ส่วน align จะจัดการในรูปข้อความแบบแนวนอน
4.position คืออะไร และ มีอะไรบ้าง Position คือ ตำแหน่งของ element
ประเภทของ Position
static: เป็นค่าพื้นฐานของทุกบล็อกโดนจะเลื่อนตามหน้าเว็บปกติ
relative: การเลื่อนออกจากตำแหน่งเดิม ตามที่เรากำหนดการดัน block จากทาง ซ้าย ขวา บน ล่าง
fixed: กำหนดให้ block นั้นยึดอยู่กับที่บนหน้าเว็บ
absolute: การเลื่อนออกจากมุมซ้ายบนสุดของหน้าจะ ตามการกำหนดการดัน block จากทาง ซ็าย ขวา บน ล่าง
sticky: การรวม relative กับ fixed ไว้ด้วยกัน โดยการกำหนด relative เพิ่มว่าจะให้ไป fix ที่ top หรือ bottom ทำให้เราเห็น block นั้นเลื่อนและถูกตรีงไว้เมื่อเรา เลื่อน scroll เมาส์
5.z-index คืออะไรใช้อย่างไร z-index จะจัดการ element ในรูปแบบของ layer มันจะคล้ายๆ Bring to back and front ใน Microsoft Office
